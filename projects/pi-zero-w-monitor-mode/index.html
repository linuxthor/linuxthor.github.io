Pi Zero W Monitor Mode

A short writeup of how to get wireless monitor mode working on the Raspberry Pi Zero W

Recently I bought a Rasperry Pi Zero W from Pimoroni and was excited to discover that the wireless chipset is the same CYW43438 wireless chip as Raspberry Pi 3, providing 802.11 wireless and bluetooth 4.0 connectivity. I've previously had success using nexmon on the Pi 3 to enable monitor mode with radiotap headers and construct a portable wireless analyser (which you can read about here.
